Este projeto tem como objetivo desenvolver um banco de dados para um sistema de controle de acesso e monitoramento de segurança, permitindo o gerenciamento de usuários, permissões, dispositivos e registros de acesso.
O sistema foi modelado seguindo as etapas de:

Modelo Conceitual (DER)
Modelo Lógico
Modelo Físico (MySQL)

*OBJETIVOS*
Controlar o acesso de usuários a locais e dispositivos
Registrar eventos e logs de acesso
Monitorar dispositivos de segurança
Aplicar conceitos de modelagem de banco de dados e boas práticas
